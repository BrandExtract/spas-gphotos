# spas-gphotos
Google Photos API helpers for SPAS/BundleBee

## Installation

### Installing spas
``` bash
$ [sudo] npm install spas -g
```

### Installing spas-gphotos
``` bash
$ [sudo] npm install spas-gphotos
```
